# Gambit fixture corpus

`gambit-16.6.0.json` is generated by `scripts/gen_fixtures.py` from the six catalog games, three known degenerate cases, and 30 deterministic random games. Its manifest records the source seed, Gambit version, immutable local image ID, case count, and SHA-256 digest.

The generator evaluates each game in a disposable container with `--network=none`, no bind mounts, a read-only root filesystem, all Linux capabilities dropped, `no-new-privileges`, a non-root user, resource limits, and an in-memory `/tmp`. Inputs enter only through standard input and output returns only through standard output. The image build downloads public Gambit v16.6.0 sources and Debian build packages; it does not receive repository content.

Build the pinned local image, then generate the corpus:

```sh
docker build --tag gt-sandbox/gambit-enummixed:16.6.0 tools/gambit
python3 scripts/gen_fixtures.py
```

The generated output is an external cross-oracle, not a replacement for the engine's own hand-authored catalog oracles and independent verifier.
